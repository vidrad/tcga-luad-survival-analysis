# tcga-luad-survival-analysis
Bioinformatics pipeline using Python (pandas, lifelines) to analyze lung adenocarcinoma (LUAD) genomic data from TCGA via UCSC Xena. Cross-references 194k+ somatic mutations with survival data for 721 patients. Identified RBM10 mutations as a significant biomarker for worse overall survival (p = 0.0228) via Kaplan-Meier &amp; Log-Rank tests.
